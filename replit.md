# Static Portfolio Website

## Overview
A raw, minimalist portfolio website built with pure HTML and JavaScript. No CSS styling is used - the site displays with browser default styles only.

## Features
- **About Me Section**: Short personal description
- **Skills Section**: List of technical skills with real-time search functionality
- **Portfolio Section**: List of projects with descriptions
- **Contact Section**: Contact information
- **Social Media Section**: Links to social profiles

## Technology Stack
- HTML5
- Vanilla JavaScript (for skills search)
- No CSS (intentionally raw/unstyled)
- No frameworks or libraries

## Structure
- `index.html` - Main portfolio page with all sections and JavaScript search functionality
- `server.py` - Simple Python HTTP server to serve the static site
